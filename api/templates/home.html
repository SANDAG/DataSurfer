USAGE

http://datasurfer.sandag.org/api/<datasource>/<year>/<geography>/<zone>/<dataset>

For example:
http://datasurfer.sandag.org/api/estimates/2013/juridiction/oceanside/ethnicity
This URL will select 2013 Estimates ethnicity estimates for the city of Oceanside.
